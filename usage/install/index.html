Install

Talks about: install

Use the following locations in order to download the binary distribution of ilo:

  • https://github.com/metio/ilo/releases

None of the widely used package managers have ilo in their official repositories yet. In case you want to help package ilo for your preferred operating system, take a look at the open packaging issues for Linux, MacOS, and Windows.

Linux

Manual

The Linux distribution of ilo is a native executable and does not require any additional software to be installed.

$ cd /path/to/ilo/installation/folder
$ curl --location https://github.com/metio/ilo/releases/download/2020.10.12/ilo-2020.10.12-linux.zip --output ilo.zip
$ unzip ilo.zip
$ chmod +x ilo-2020.10.12/ilo
$ ln --symbolic --relative ilo-2020.10.12/ilo ~/.local/bin/ilo

Fedora

Fedora users can install ilo from a COPR repository like this:

$ sudo dnf copr enable sebhoss/ilo
$ sudo dnf install ilo

Fedora Silverblue

Fedora Silverblue users can install ilo from the same COPR by following these steps:

$ curl --location https://copr.fedorainfracloud.org/coprs/sebhoss/ilo/repo/fedora-$(rpm -E %fedora)/sebhoss-ilo-fedora-$(rpm -E %fedora).repo --output ilo.repo
$ sudo mv ilo.repo /etc/yum.repos.d/ilo.repo
$ rpm-ostree install ilo

MacOS

The MacOS distribution of ilo is a native executable and does not require any additional software to be installed.

$ cd /path/to/ilo/installation/folder
$ curl --location https://github.com/metio/ilo/releases/download/2020.10.12/ilo-2020.10.12-mac.zip --output ilo.zip
$ unzip ilo.zip
$ chmod +x ilo-2020.10.12/ilo
$ ln -s /path/to/ilo/installation/folder/ilo-2020.10.12/ilo /usr/local/bin/ilo
# in case the above gives you a permission denied error, execute the same command with sudo, e.g. 'sudo !!'
# in order to fix permissions run: sudo chmod -R u=rwX,go=rX /usr/local/bin

Windows

The Windows distribution of ilo is a native executable and does not require any additional software to be installed.

# PowerShell
$ New-Item -Path $env:USERPROFILE -Name "bin" -ItemType "directory" # create directory if it does not exit
$ control sysdm.cpl # manually add '$env:USERPROFILE\bin' to the PATH of your user account
$ echo $env:PATH # ensure that the 'bin' directory is in your PATH
$ Set-Location -Path C:\path\to\ilo\installation\folder
$ Invoke-RestMethod -Uri https://github.com/metio/ilo/releases/download/2020.10.12/ilo-2020.10.12-windows.zip -OutFile "ilo.zip"
$ Expand-Archive -Path "ilo.zip" -DestinationPath "."
$ Copy-Item -Path "ilo-2020.10.12\ilo.exe" -Destination "$env:USERPROFILE\bin\ilo.exe"

Other

For all other platforms that support at least Java 11, run something like this:

$ cd /path/to/ilo/installation/folder
$ curl --location https://github.com/metio/ilo/releases/download/2020.10.12/ilo-2020.10.12-java11.zip --output ilo.zip
$ unzip ilo.zip
$ chmod +x ilo-2020.10.12/ilo
# move 'ilo' start script to your preferred location
