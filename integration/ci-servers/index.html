CI Servers

Talks about: jenkins

Jenkins

Jenkins knows multiple ways to define jobs. In order to integrate with ilo make sure the ilo is installed on your slaves.

Declarative Pipeline

pipeline {
  stages {
    stage('Build with ilo') {
      sh 'ilo @build-things'
    }
  }
}

build-things is an arguments file.
